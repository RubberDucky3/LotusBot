LotusBot

LotusBot is a multifunctional Discord bot for League of Legends and Rubik's Cube enthusiasts.

Features

  • League of Legends: Fetch summoner stats, match history, and in-game summaries.
  • Rubik's Cube: Generate scrambles and compute optimal solutions using the Kociemba algorithm.

This bot is designed for educational and entertainment purposes, making it easy for users to access game stats and puzzle-solving tools without leaving Discord.
